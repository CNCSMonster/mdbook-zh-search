## 实现细节

### 英文处理改进

由于我们禁用了默认的 Pipeline（包含 trimmer），我们在 tokenize 函数中自己处理标点符号：

```rust
// 对英文文本的处理
text.split(|c: char| c.is_whitespace() || c == '-')
    .filter(|s| !s.is_empty())
    .map(|s| {
        // 去除首尾标点，但保留内部标点（如 println!）
        s.trim_matches(|c: char| c.is_ascii_punctuation() && c != '_' && c != '!')
          .to_lowercase()
    })
    .filter(|s| !s.is_empty() && s.len() <= MAX_WORD_LENGTH_TO_INDEX)
    .collect()
```

### 特殊处理

- **保留 `_`**: 支持编程标识符（如 `my_variable`）
- **保留 `!`**: 支持 Rust 宏（如 `println!`）
- **去除首尾标点**: 匹配原 Pipeline trimmer 的行为
