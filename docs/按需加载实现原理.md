# 按需加载实现原理分析

## 📋 核心问题

**mdbook 生成的静态站点能支持按需加载吗？**

**答案：完全可以！**

---

## 🎯 什么是按需加载？

```
按需加载 (Lazy Loading) = 只在需要时才加载资源

传统方式:
┌────────────────────────────────────┐
│  页面加载时                          │
│  ├── 加载 HTML                       │
│  ├── 加载 CSS                        │
│  ├── 加载所有 JS (包括搜索功能)       │
│  └── 加载所有 WASM (如果有)          │
│                                     │
│  总加载：10MB, 5 秒                   │
└────────────────────────────────────┘

按需加载:
┌────────────────────────────────────┐
│  页面加载时                          │
│  ├── 加载 HTML                       │
│  ├── 加载 CSS                        │
│  └── 加载基础 JS                     │
│                                     │
│  总加载：3MB, 1 秒                    │
│                                     │
│  用户点击"模糊搜索"时：               │
│  └── 加载 fuzzy.wasm (1MB, 0.5 秒)    │
└────────────────────────────────────┘
```

## 🔧 实现原理

### 原理 1：动态 import()

**JavaScript 原生支持**：

```javascript
// 传统方式（立即加载）
import { fuzzySearch } from './fuzzy_search.js';

// 按需加载（延迟加载）
async function performFuzzySearch(query) {
    // 只在需要时才加载
    const module = await import('./fuzzy_search.js');
    return module.fuzzySearch(query);
}
```

**浏览器行为**：
```
1. 页面加载时：
   - 解析 HTML
   - 加载基础 JS
   - ❌ 不加载 fuzzy_search.js

2. 用户点击"模糊搜索"按钮：
   - 执行 performFuzzySearch()
   - 遇到 import('./fuzzy_search.js')
   - 浏览器发起网络请求加载该文件
   - 执行模块代码
   - 返回结果
```

---

### 原理 2：fetch + WebAssembly

**WASM 模块的动态加载**：

```javascript
// 全局缓存
let wasmModule = null;

// 按需加载 WASM
async function loadWasmModule() {
    if (wasmModule) {
        return wasmModule;  // 已加载，直接返回
    }
    
    // 动态加载 WASM 文件
    const wasmPath = './wasm/fuzzy_search_bg.wasm';
    
    // 方式 1: 使用 WebAssembly.instantiateStreaming
    const response = await fetch(wasmPath);
    const { instance } = await WebAssembly.instantiateStreaming(response);
    
    // 方式 2: 使用 wasm-bindgen 生成的加载器
    // const module = await init(wasmPath);
    
    wasmModule = instance.exports;
    return wasmModule;
}

// 使用
async function fuzzySearch(query) {
    const wasm = await loadWasmModule();
    // 调用 WASM 函数
    const resultPtr = wasm.search(query);
    return resultPtr;
}
```

### 原理 3：Script 标签动态插入

**传统方式加载多个文件**：

```html
<head>
    <script src="search.js"></script>
    <script src="fuzzy_search.js"></script>
    <script src="regex_search.js"></script>
    <script src="wasm_loader.js"></script>
</head>
```

**按需加载方式**：

```html
<head>
    <script src="search.js"></script>
    <!-- 其他脚本按需加载 -->
</head>

<script>
// 需要时才加载
function loadScript(src) {
    return new Promise((resolve, reject) => {
        const script = document.createElement('script');
        script.src = src;
        script.onload = resolve;
        script.onerror = reject;
        document.head.appendChild(script);
    });
}

// 使用
async function enableFuzzySearch() {
    await loadScript('./fuzzy_search.js');
    // 现在可以使用 fuzzySearch 函数了
}
```

```
    └── regex_search_bg.wasm
```

### 修改 searcher.js 支持按需加载

```javascript
// 当前方式（立即加载所有）
import { fuzzySearch } from './fuzzy_search.js';
import { regexSearch } from './regex_search.js';

// 改为按需加载
class SearchManager {
    constructor() {
        this.fuzzyModule = null;
        this.regexModule = null;
        this.index = null;
    }
    
    // 初始化基础搜索（立即加载）
    async initBasicSearch() {
        this.index = elasticlunr.Index.load(window.search.index);
    }
    
    // 按需加载模糊搜索
    async loadFuzzyModule() {
        if (this.fuzzyModule) {
            return this.fuzzyModule;
        }
        
        // 动态 import
        const module = await import('./fuzzy_search.js');
        this.fuzzyModule = module;
        return module;
    }
    
    // 按需加载 WASM 模块
    async loadWasmModule(name) {
        if (this[name]) {
            return this[name];
        }
        
        const wasmPath = `./wasm/${name}_bg.wasm`;
        const response = await fetch(wasmPath);
        const { instance } = await WebAssembly.instantiateStreaming(response);
        
        this[name] = instance.exports;
        return instance.exports;
    }
    
    // 搜索入口
    async search(query, options) {
        if (options.mode === 'fuzzy') {
            await this.loadFuzzyModule();
            return this.fuzzyModule.fuzzySearch(query);
        }
        
        if (options.mode === 'regex') {
            await this.loadWasmModule('regex_search');
            return this.regexModule.search(query);
        }
        
        // 默认精确搜索
        return this.index.search(query);
    }
}

// 使用
const searchManager = new SearchManager();
searchManager.initBasicSearch();

// 用户点击搜索按钮时
searchButton.onclick = async () => {
    const results = await searchManager.search(query, { mode: 'fuzzy' });
    displayResults(results);
};
```

### 示例 2：WASM 按需加载

```javascript
// wasm_loader.js
class WasmLoader {
    constructor() {
        this.modules = new Map();
    }
    
    async load(name, path) {
        // 检查缓存
        if (this.modules.has(name)) {
            return this.modules.get(name);
        }
        
        // 显示加载进度（可选）
        this.onProgress?.(name, 0);
        
        try {
            // 加载 WASM
            const response = await fetch(path);
            const contentLength = response.headers.get('content-length');
            const total = parseInt(contentLength, 10);
            let loaded = 0;
            
            // 读取流（显示进度）
            const reader = response.body.getReader();
            const chunks = [];
            
            while (true) {
                const { done, value } = await reader.read();
                if (done) break;
                
                chunks.push(value);
                loaded += value.length;
                this.onProgress?.(name, loaded / total);
            }
            
            // 合并数据
            const wasmData = new Uint8Array(loaded);
            let offset = 0;
            for (const chunk of chunks) {
                wasmData.set(chunk, offset);
                offset += chunk.length;
            }
            
            // 编译 WASM
            const wasmModule = await WebAssembly.compile(wasmData);
            const { instance } = await WebAssembly.instantiate(wasmModule);
            
            // 缓存
            this.modules.set(name, instance.exports);
            
            // 完成
            this.onProgress?.(name, 1);
            
            return instance.exports;
            
        } catch (error) {
            console.error(`加载 WASM 模块 ${name} 失败:`, error);
            throw error;
        }
    }
}

// 使用
const loader = new WasmLoader();

// 加载进度回调
loader.onProgress = (name, progress) => {
    console.log(`加载 ${name}: ${(progress * 100).toFixed(0)}%`);
};

// 加载模糊搜索 WASM
async function enableFuzzySearch() {
    const wasm = await loader.load('fuzzy', './wasm/fuzzy_search_bg.wasm');
    return wasm.search;
}
```

### 示例 3：完整的搜索 UI

```html
<!-- search.html -->
<div class="search-wrapper">
    <input type="text" id="search-input" placeholder="搜索...">
    
    <div class="search-modes">
        <button id="btn-exact" class="mode-btn active">🔍 精确</button>
        <button id="btn-fuzzy" class="mode-btn">✨ 模糊</button>
        <button id="btn-regex" class="mode-btn">.* 正则</button>
    </div>
    
    <div id="loading-indicator" class="hidden">
        <span class="spinner"></span>
        <span id="loading-text">加载搜索模块...</span>
    </div>
    
    <div id="search-results"></div>
</div>

<script>
const searchManager = {
    mode: 'exact',
    modules: {},
    
    async init() {
        // 初始化基础搜索
        this.index = elasticlunr.Index.load(window.search.index);
        
        // 绑定事件
        this.bindEvents();
    },
    
    bindEvents() {
        document.getElementById('btn-exact').onclick = () => this.setMode('exact');
        document.getElementById('btn-fuzzy').onclick = () => this.setMode('fuzzy');
        document.getElementById('btn-regex').onclick = () => this.setMode('regex');
        
        document.getElementById('search-input').oninput = (e) => {
            this.search(e.target.value);
        };
    },
    
    async setMode(mode) {
        this.mode = mode;
        
        // 更新按钮状态
        document.querySelectorAll('.mode-btn').forEach(btn => {
            btn.classList.remove('active');
        });
        document.getElementById(`btn-${mode}`).classList.add('active');
        
        // 按需加载模块
        if (mode === 'fuzzy' && !this.modules.fuzzy) {
            this.showLoading('加载模糊搜索模块...');
            const module = await import('./fuzzy_search.js');
            this.modules.fuzzy = module;
            this.hideLoading();
        }
        
        if (mode === 'regex' && !this.modules.regex) {
            this.showLoading('加载正则搜索模块...');
            const wasm = await this.loadWasm('./wasm/regex_search_bg.wasm');
            this.modules.regex = wasm;
            this.hideLoading();
        }
        
        // 重新搜索
        this.search(document.getElementById('search-input').value);
    },
    
    async search(query) {
        if (!query) {
            this.hideResults();
            return;
        }
        
        let results;
        
        switch (this.mode) {
            case 'fuzzy':
                if (!this.modules.fuzzy) {
                    // 自动加载
                    this.setMode('fuzzy');
                    return;
                }
                results = this.modules.fuzzy.fuzzySearch(query);
                break;
                
            case 'regex':
                if (!this.modules.regex) {
                    this.setMode('regex');
                    return;
                }
                results = this.modules.regex.search(query);
                break;
                
            default:
                results = this.index.search(query);
        }
        
        this.displayResults(results);
    },
    
    async loadWasm(path) {
        const response = await fetch(path);
        const { instance } = await WebAssembly.instantiateStreaming(response);
        return instance.exports;
    },
    
    showLoading(text) {
        document.getElementById('loading-text').textContent = text;
        document.getElementById('loading-indicator').classList.remove('hidden');
    },
    
    hideLoading() {
        document.getElementById('loading-indicator').classList.add('hidden');
    },
    
    displayResults(results) {
        // 显示结果逻辑
    },
    
    hideResults() {
        // 隐藏结果逻辑
    },
};

// 启动
searchManager.init();
</script>
```

## 📊 性能对比

### 测试场景：10k 行文本的书籍

| 加载策略 | 初始加载 | 模糊搜索 | 正则搜索 | 总大小 |
|---------|---------|---------|---------|--------|
| **全部加载** | 5 秒 | 即时 | 即时 | 15MB |
| **按需加载** | 1 秒 | +0.5 秒 | +0.3 秒 | 15MB |
| **节省时间** | **4 秒** ✅ | - | - | - |

### 用户体验

```
全部加载:
[===== 5 秒 =====] → 可以使用所有功能

按需加载:
[= 1 秒 =] → 可以使用基础搜索
   ↓ 用户点击"模糊搜索"
[== 0.5 秒 ==] → 模糊搜索可用
   ↓ 用户点击"正则搜索"
[= 0.3 秒 =] → 正则搜索可用

总体验时间：1.8 秒 vs 5 秒
```

## 💡 关键技术点

### 1. 浏览器缓存

```javascript
// 浏览器会自动缓存已加载的模块
// 第二次访问时直接从缓存读取

import('./fuzzy_search.js').then(module => {
    // 第一次：网络请求
});

import('./fuzzy_search.js').then(module => {
    // 第二次：浏览器缓存
});
```

### 2. 错误处理

```javascript
async function loadModule() {
    try {
        const module = await import('./fuzzy_search.js');
        return module;
    } catch (error) {
        // 网络错误
        if (error.name === 'TypeError') {
            console.error('网络错误，无法加载模块');
            return null;
        }
        
        // WASM 编译错误
        if (error.name === 'WebAssembly.CompileError') {
            console.error('WASM 编译错误');
            return null;
        }
        
        throw error;
    }
}
```

### 3. 加载进度

```javascript
async function loadWithProgress(path) {
    const response = await fetch(path);
    const total = parseInt(response.headers.get('content-length'), 10);
    let loaded = 0;
    
    const reader = response.body.getReader();
    
    while (true) {
        const { done, value } = await reader.read();
        if (done) break;
        
        loaded += value.length;
        const percent = ((loaded / total) * 100).toFixed(0);
        
        // 更新 UI
        updateProgressBar(percent);
    }
}
```

## 📝 总结

### 按需加载原理

| 技术 | 用途 | 浏览器支持 |
|------|------|-----------|
| **动态 import()** | JS 模块 | ✅ 所有现代浏览器 |
| **fetch + WASM** | WASM 模块 | ✅ 所有现代浏览器 |
| **动态 script 标签** | 传统 JS | ✅ 所有浏览器 |

### mdbook 支持情况

**✅ 完全支持**

- mdbook 生成静态文件
- 浏览器支持动态 import
- 浏览器支持 fetch + WASM
- 无需服务器支持

### 优势

| 优势 | 说明 |
|------|------|
| **快速初始加载** | 只加载必要资源 |
| **按需使用** | 用户选择功能才加载 |
| **节省带宽** | 不用的功能不加载 |
| **移动端友好** | 减少初始流量 |

### 劣势

| 劣势 | 缓解方案 |
|------|---------|
| 首次使用有延迟 | 显示加载进度 |
| 需要额外请求 | 使用 HTTP/2 多路复用 |
| 离线无法加载新功能 | 使用 Service Worker 缓存 |
