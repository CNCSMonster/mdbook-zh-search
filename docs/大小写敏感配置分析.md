# 大小写敏感配置分析

## 📋 核心问题

1. **当前是否大小写敏感？**
2. **是否需要配置？**
3. **与模糊/正则搜索是否冲突？**

---

## 🔍 当前行为分析

### elasticlunr 默认行为

**答案：当前是不区分大小写的**

```javascript
// elasticlunr 默认配置
const index = elasticlunr(function() {
    this.addField('title');
    this.addField('body');
    this.setRef('id');

// 搜索："RUST" → "rust" → 匹配 ✅
// 搜索："rust" → "rust" → 匹配 ✅
```

## 📊 测试验证

### 当前 mdbook-zh-search 的行为

```rust
// crates/mdbook-html/src/html_handlebars/search.rs

fn tokenize(text: &str) -> Vec<String> {
    if is_cjk_text(text) {
        // 中文：转小写
        JIEBA.cut_for_search(text, false)
            .into_iter()
            .map(|s| s.to_lowercase())  // ✅ 转小写
            .collect()
    } else {
        // 英文：转小写
        text.split_whitespace()
            .map(|s| {
                s.trim_matches(|c: char| c.is_ascii_punctuation() && c != '_' && c != '!')
                  .to_lowercase()  // ✅ 转小写
            })
            .collect()
    }
}
```

**结论：当前构建时和运行时都不区分大小写**

## 🎯 是否需要大小写敏感配置？

### 场景分析

#### 场景 1：搜索普通英文单词

```
文档："The Rust Programming Language"
搜索："rust" → 找到 "Rust" ✅
搜索："RUST" → 找到 "Rust" ✅
搜索："Rust" → 找到 "Rust" ✅

需求：不区分大小写 ✅ 当前行为
```

#### 场景 2：搜索专有名词/缩写

```
文档："使用 HTTP 协议"
搜索："http" → 找到 "HTTP" ✅ (不区分)
搜索："HTTP" → 找到 "HTTP" ✅ (不区分)

需求：不区分大小写 ✅ 当前行为
```

#### 场景 3：搜索代码标识符

```
文档："fn GetValue() { ... }"
搜索："GetValue" → 找到 "GetValue" ✅
搜索:"getvalue" → 找到 "GetValue" ⚠️ (可能不是想要的)
搜索:"GETVALUE" → 找到 "GetValue" ⚠️ (可能不是想要的)

需求：区分大小写 ❌ 当前不支持
```

#### 场景 4：搜索变量名

```
文档："let UserName = ...;"
搜索："UserName" → 找到 ✅
搜索："username" → 找到 ⚠️ (可能是不同的变量)

需求：区分大小写 ❌ 当前不支持
```

## 💡 设计方案

### 方案 A：不添加大小写敏感配置（推荐）

**理由**:
- 90% 的场景不需要
- 保持简单，用户友好
- 减少配置复杂度
- 与 Google/百度等搜索引擎一致

**实现**:
```javascript
// 保持当前行为，不区分大小写
const options = {
    bool: 'AND',
    expand: true,
    fields: {
        title: {boost: 2},
        body: {boost: 1},
    },
};
```

### 方案 B：添加大小写敏感配置（可选）

**理由**:
- 满足代码搜索需求
- 高级功能，可选配置
- 与正则搜索配合使用

**实现**:

## 🔧 与高级功能的关系

### 大小写敏感 vs 模糊搜索

**关系：不冲突，正交配置**

```
模糊搜索 + 不区分大小写:
  搜索："getvalue" → 找到 "GetValue", "GETVALUE" ✅

模糊搜索 + 区分大小写:
  搜索："GetValue" → 找到 "GetValue" ✅
  搜索："getvalue" → 无结果 ✅
```

**实现**:
```javascript
// Fuse.js 支持两个独立配置
const fuse = new Fuse(documents, {
    isCaseSensitive: false,  // 大小写敏感
    threshold: 0.3,          // 模糊程度
});
```

---

### 大小写敏感 vs 正则搜索

**关系：正则表达式本身支持大小写敏感**

```javascript
// 不区分大小写
/abc/i  // 匹配 "abc", "ABC", "Abc"

// 区分大小写
/abc/   // 只匹配 "abc"
```

**实现**:

## 📝 完整实现（如果需要）

### searcher.js

```javascript
class SearchManager {
    constructor() {
        this.mode = 'exact';
        this.caseSensitive = false;
        this.modules = {};
    }
    
    toggleCaseSensitive() {
        this.caseSensitive = !this.caseSensitive;
        this.updateCaseButton();
        this.search(this.searchInput.value);
    }
    
    updateCaseButton() {
        const btn = document.getElementById('btn-case');
        if (this.caseSensitive) {
            btn.classList.add('active');
            btn.title = '大小写敏感 (开启)';
        } else {
            btn.classList.remove('active');
            btn.title = '大小写敏感 (关闭)';
        }
    }
    
    async search(query) {
        let results;
        
        switch (this.mode) {
            case 'fuzzy':
                // Fuse.js 支持大小写敏感
                this.modules.fuzzy.options.isCaseSensitive = this.caseSensitive;
                results = this.modules.fuzzy.search(query);
                break;
                
            case 'regex':
                // 正则表达式通过 flag 控制
                results = this.regexSearch(query, this.caseSensitive);
                break;
                
            default:
                // elasticlunr 不支持，需要后处理
                results = this.exactSearch(query, this.caseSensitive);
        }
        
        this.displayResults(results);
    }
    
    exactSearch(query, caseSensitive) {
        // elasticlunr 默认转小写搜索
        const results = this.index.search(query.toLowerCase(), options);
        
        // 如果需要区分大小写，后处理过滤
        if (caseSensitive) {
            return results.filter(result => {
                const doc = this.index.documentStore.docs[result.ref];
                const allText = `${doc.title} ${doc.body}`;
                return allText.includes(query);  // 原始大小写匹配
            });
        }
        
        return results;
    }
    
    regexSearch(query, caseSensitive) {
        const flags = caseSensitive ? 'g' : 'gi';
        const regex = new RegExp(query, flags);
        
        const docs = this.index.documentStore.docs;
        const results = [];
        
        for (const [ref, doc] of Object.entries(docs)) {
            const allText = `${doc.title} ${doc.body}`;
            const matches = allText.match(regex);
            
            if (matches) {
                results.push({
                    ref,
                    score: matches.length,
                    doc,
                    matches: matches.slice(0, 5),
                });
            }
        }
        
        return results.sort((a, b) => b.score - a.score);
    }
}
```
