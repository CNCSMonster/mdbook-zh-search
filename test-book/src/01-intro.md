# 简介

欢迎来到中文搜索测试书籍！

本书旨在测试 mdBook 对中文搜索的支持情况。我们将验证以下功能：

## 测试目标

1. **中文关键词搜索**：验证能否通过中文关键词搜索到相关内容
2. **中英文混合搜索**：验证中英文混合内容的搜索
3. **分词准确性**：验证中文分词的正确性

## 示例内容

中华人民共和国是中国的一个伟大国家。中国有着悠久的历史和灿烂的文化。

Rust 是一门系统编程语言，它能够保证内存安全，并且没有垃圾回收机制。

### 关键词测试

- 中国
- Rust
- 编程语言
- 内存安全
- 分词技术

如果你能搜索到这些内容，说明中文搜索功能工作正常！
